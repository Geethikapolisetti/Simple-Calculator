# Simple Calculator

This is a calculator I built using HTML, CSS and JavaScript while practicing frontend concepts and improving UI design.
I started with a basic calculator, then gradually added features like theme switching, history, and better styling to make it feel more like a real app.

## Live Demo

https://geethikapolisetti.github.io/Simple-Calculator/

## What it can do

* Perform basic operations (+, −, ×, ÷)
* Includes some scientific functions (like sqrt, sin, log, etc.)
* Toggle between dark mode and light mode
* Stores calculation history (using localStorage)
* Works with keyboard input as well
* 
## Why I made this

I wanted to go beyond a simple calculator and try:

* handling user input properly
* managing UI state (theme toggle)
* using localStorage in a real use case
* writing cleaner JavaScript instead of just using eval everywhere

## Tech used

* HTML
* CSS (used variables for theming)
* JavaScript

## How to run

Just download or clone the repo and open `index.html` in your browser.
